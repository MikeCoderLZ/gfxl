Detailed Description

Class for expressing, storing, and comparing library versions.

Specifically for OpenGL version checks, but can be used for any library. Appropriate operators are overloaded so that version numbers can be compared naturally in code. There is no default constructor; all version objects must pertain to a specific version number.
