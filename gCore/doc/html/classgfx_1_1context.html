Detailed Description

Represents an OpenGL rendering context.

The way in which contexts and windows interact is obscure; It is possible that some OpenGL settings need to be saved in order to make this all work correctly.

The context class is a representation of an OpenGL rendering context. A valid context and associated window are needed to render using OpenGL, and an initialized video system must be present. Some tedious details are handled automagically and configuration uses the settings member class.
