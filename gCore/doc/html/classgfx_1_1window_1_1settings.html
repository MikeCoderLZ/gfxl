Detailed Description

The settings object for the window class. Mutators can be chained.

Constructor & Destructor Documentation

gfx::window::settings::settings ( )
inline

Construct a default settings object.

The default values for a window are essentially useless, consisting of the bare necessary graphical features and zeroed out dimensions.
