Detailed Description

The base class for channel format selectors.

Channel format selectors are a utility class used in the configuration of textures. Functionally, there are an enum with special type charactersitics to make formatting errors cause compilation errors.

They consist of a multiply inheritted class hierarchy; the derived classes at the end of the hierarchy inherit from the directly derived classes of bit_t. The instances of these terminal classes which are provided are used to select the bit depth of the chanels in the texture, being passed to the number format and chanel multplicity selectors in the settings class.

Using multiple inheritance here means that only combinations of channel bit depth, number format, and chanel mulplicity which are legal in OpenGL are allowed. Illegal combinations cause compilation errors, stopping a developer from making a mistake and causing error messages that more or less actually mean what they say.
